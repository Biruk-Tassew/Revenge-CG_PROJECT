# Revenge-CG_PROJECT
Revenge - In this simple game,  dinosaurs seek vengeance on humans who are to blame for their extinction. 

# How to play?
just run Revenge.py
you can also use the command:

    python Revenge.py
  
  To jump the logs just press the up key.
